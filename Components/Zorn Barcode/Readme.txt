Welcome to BarComp version 3.0
==============================
Barcomp is a set of four Barcode Components for Delphi. Two of them
are to be used with QuickReport, the other two are for general use. 

The components can be used with Delphi 2, 3, 4, 5 and 6 and all
versions of QuickReport from 2.0.

IMPORTANT
=========
Users of Delphi 6: in addition to the normal installation procedure, 
the file 'designide.dcp' should be added to the Required Packages
section of the package, otherwise an error will occur.

C++ Builder users: the non-QuickReport components can be installed
without problems (normally). The QR components are generally causing
problems with specific QR files. Maybe someone finds a solution to
this...

For more details on the installation please refer to the help file 
(barcomp.chm).

A note about printing
=====================
For best readability (by a barcode reader), use a printer with
at least 300 dpi resolution. Higher printer resolutions result
in better readability.
